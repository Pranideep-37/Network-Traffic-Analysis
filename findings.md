# Network Traffic Analysis

## Objective

The purpose of this task was to capture network traffic using Wireshark and identify common network protocols.

## Protocols Identified

### DNS

DNS packets were observed when visiting websites. These packets are used to translate domain names into IP addresses.

### TCP

TCP packets were used to establish and maintain communication between my computer and web servers.

### TLS

TLS packets were captured while accessing HTTPS websites. TLS encrypts the communication between the client and the server.

## Packet Observations

- DNS requests were generated when opening websites.
- TCP handled the connection between the client and the server.
- TLS was used to secure the data being transmitted.
- Source and destination IP addresses were visible in the packet details.

## Conclusion

This task gave me hands-on experience with Wireshark. I learned how to capture network traffic, apply protocol filters, and identify the role of different protocols in everyday internet communication.
