# Network-Traffic-Analysis
## Objective

The objective of this task was to capture live network traffic using Wireshark and identify the different network protocols used during normal internet browsing.

## Tools Used

- Wireshark
- Web Browser
- macOS

## Files Included

- `capture.pcap` - Captured network traffic.
- `findings.md` - Summary of the analysis.
- `screenshots/` - Screenshots of the packet capture and protocol filters.

## Protocols Observed

- DNS
- TCP
- TLS

## Outcome

This task helped me understand how network traffic is captured and analyzed using Wireshark. I also learned how DNS, TCP, and TLS work together during web browsing.
